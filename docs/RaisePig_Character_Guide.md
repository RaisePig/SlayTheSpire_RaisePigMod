## 角色概述

### 基本信息

| 属性 | 数值 |
|-----|------|
| 初始生命 | 75 |
| 初始能量 | 3 |
| 初始遗物 | 养猪秘籍 |
| 卡牌颜色 | 粉色 (RaisePigPink) |

### 角色定位

养猪人是一个**发育型**角色，其核心玩法围绕**能量上限**的增长展开。与其他角色追求即时收益不同，养猪人更注重长期投资，通过不断提升能量上限来解锁更强大的卡牌效果。

### 核心特色

1. **能量经济系统** - 能量上限可以永久增长，但也可以被消耗
2. **投喂-收获循环** - 在敌人身上堆叠投喂，通过收获获得奖励
3. **里程碑奖励** - 投喂达到7层/15层时触发特殊效果
4. **风险与收益** - 部分强力卡牌需要牺牲能量上限

## 核心机制详解

### 🐷 养猪 (RaisePigPower)

**类型**: 玩家Buff

**效果**: 每次恢复能量时，有 **(层数 × 10%)** 的概率永久增加1点能量上限

**触发时机**: 每回合开始恢复能量时

**策略价值**:
- 前期堆叠养猪层数，中后期稳定涨能量上限
- 概率性质意味着层数越高越稳定
- 10层养猪 = 100%触发 = 每回合必涨能量

**相关卡牌**:
| 卡牌 | 获得层数 | 稀有度 |
|-----|---------|-------|
| 蓄力打击 | 1 | BASIC |
| 蓄力防御 | 1 | BASIC |
| 养猪密码 | 3(4) | COMMON |
| 肥猪拱门 | 2(3) | COMMON |
| 猪圈扩建 | 1(2) | UNCOMMON |
| 金猪献瑞 | 3(4) | RARE |

---

### 🍖 投喂 (FeedPower)

**类型**: 敌人Debuff

**效果**: 
- 达到 **7层** 时，收获可获得 +1 能量上限
- 达到 **15层** 时，收获可额外升级牌组一张牌

**策略价值**:
- 7层是第一个重要里程碑，性价比最高
- 15层需要大量投入，但奖励极其丰厚
- 多目标战斗时可以同时养多只"猪"

**投喂效率统计**:
| 投喂量 | 代表卡牌 | 费用效率 |
|-------|---------|---------|
| 1 | 一巴掌一甜枣 | 1投喂/1费 |
| 2 | 优质饲料 | 2投喂/1费 |
| 4-5 | 撑死你 | 4-5投喂/2费 |
| 5-7 | 饲料霸主(AOE) | 5-7投喂/2费 |
| 等于伤害 | 无尽食欲 | 12-16投喂/2费 |

---

### 🌾 收获 (Harvest)

**效果**: 消耗目标身上所有投喂，触发投喂效果

**奖励机制**:
```
投喂 ≥ 7层  → 永久 +1 能量上限
投喂 ≥ 15层 → 永久 +1 能量上限 + 升级一张牌
```

**相关卡牌**:
| 卡牌 | 效果 | 稀有度 |
|-----|------|-------|
| 垦荒 | 格挡+收获 | COMMON |
| 秋收起义 | 收获(消耗) | COMMON |
| 丰收日 | 伤害+收获 | COMMON |
| 紧急出栏 | 触发收获但不消耗投喂 | UNCOMMON |
| 大规模出栏 | AOE收获 | RARE |

---

### 🥬 收菜 (KillPig)

**效果**: 清除目标所有状态，造成投喂层数的伤害，你失去等量能量

**使用时机**:
- 紧急清场时使用
- 敌人投喂层数很高时可作为斩杀
- 注意：会损失能量，需谨慎使用

## 构筑方向指南

### 🔴 投喂流 (Feed Build)

**核心理念**: 快速积累投喂层数，触发里程碑奖励

**关键卡牌**:
- 疯狂投喂 - 攻击牌额外投喂
- 催肥计划 - 投喂翻倍
- 精细化养殖 - 全局投喂+1
- 无尽食欲 - 伤害=投喂
- 杂食狂暴 - 3次攻击，每次投喂2

**优势**: 
- 可以快速触发15层奖励
- 力量加成后投喂量恐怖
- 配合圈养卡牌爆发伤害高

**劣势**:
- 需要时间堆叠
- 单目标战斗效率高，多目标分散

**推荐搭配**:
```
核心: 精细化养殖 + 疯狂投喂
输出: 无尽食欲 + 猪王之怒
辅助: 催肥计划 + 骑猪冲锋
```

---

### 🟢 发育流 (Scaling Build)

**核心理念**: 专注提升能量上限，解锁强力效果

**关键卡牌**:
- 养猪大亨 - 每回合+1能量上限
- 投资理财(固有) - 稳定起手
- 农场经理 - 能量上限奖励
- 猪队友 - 超额能量上限收益
- 飞猪突袭 - 发育后变正常牌

**能量上限收益卡牌**:
| 卡牌 | 条件 | 额外效果 |
|-----|------|---------|
| 黑猪掌 | ≥5 | 双倍伤害 |
| 飞猪突袭 | ≥6 | 失去虚无 |
| 猪倌的智慧 | ≥6 | +1抽牌 |
| 农场经理 | ≥5/6 | +1能量 |
| 猪队友 | >3 | 每点+3格挡/+1抽/+1能量 |

**推荐搭配**:
```
核心: 养猪大亨 + 农场经理
起手: 投资理财(固有)
输出: 黑猪掌 + 飞猪突袭
防御: 猪队友 + 猪脂护盾
```

---

### 🔵 收获流 (Harvest Build)

**核心理念**: 频繁使用收获牌触发奖励

**关键卡牌**:
- 紧急出栏 - 触发收获但不消耗投喂(升级后可复用)
- 猪肉期货 - 收获时获得格挡
- 永恒农场 - 收获时获得养猪
- 猪突猛击 - 每次收获+3永久伤害

**循环策略**:
1. 保持敌人投喂在7层左右
2. 使用紧急出栏(升级)反复触发收获效果
3. 收获不消耗投喂 → 可持续触发

**推荐搭配**:
```
核心: 紧急出栏+ + 永恒农场
投喂: 规模化 + 猪鼻拱击
辅助: 猪肉期货 + 猪突猛击
```

---

### 🟡 爆发流 (Burst Build)

**核心理念**: 积累后一波带走

**关键卡牌**:
- 猪市崩盘(消耗) - AOE清场
- 终极丰收 - 投喂总量×倍率
- 猪帝降临 - 15层投喂伤害翻倍
- 碎猪存钱罐(消耗) - 牺牲发育换爆发

**爆发组合**:
```
准备阶段: 对所有敌人投喂 → 饲料霸主、放猪觅食
爆发阶段: 终极丰收 或 猪市崩盘
斩杀阶段: 猪帝降临(15层翻倍)
```

---

### 🟣 防御发育流 (Defensive Scaling)

**核心理念**: 不攻击回合触发猪息，稳步发育

**关键卡牌**:
- 养精蓄锐(保留) - 猪息+1能量上限
- 泥潭浴 - 猪息+养猪
- 猪圈扩建 - 格挡+养猪
- 猪脂护盾 - 能量上限转防御

**猪息触发条件**: 本回合不打出攻击牌

**策略**:
1. 保留养精蓄锐到安全回合
2. 该回合只打技能/能力牌
3. 回合结束触发猪息，永久+1能量上限
4. 用高格挡撑过不攻击的回合

## Boss战策略

### 第一幕Boss

**史莱姆Boss**:
- 分裂前快速堆叠投喂
- 分裂后用AOE投喂卡处理小史莱姆
- 推荐带：饲料溅射、放猪觅食

**六角虫**:
- 发育时间充足
- 可以安全堆叠养猪层数
- 利用它的弱点回合收获

**守护者**:
- 需要格挡对策
- 模式切换时趁机投喂
- 推荐带：猪圈扩建、泥潭浴

---

### 第二幕Boss

**收藏家**:
- 先处理火把
- 对主体堆叠投喂
- 收获时机在补充火把前

**冠军**:
- 半血后伤害增加
- 提前堆叠投喂到15层
- 一波带走避免持久战

**自动机**:
- 多目标，适合AOE投喂
- 饲料霸主效率极高
- 终极丰收/猪市崩盘清场

---

### 第三幕Boss

**觉醒者**:
- 两阶段，有足够发育时间
- 第一阶段稳步堆叠
- 第二阶段爆发

**时间吞噬者**:
- 12牌惩罚需注意
- 选择高效率卡牌
- 养猪大亨长期战斗收益大

**噩梦之心** (第四幕):
- 终极考验
- 需要完整构筑
- 推荐带终极丰收/猪帝降临

## 总结

养猪人是一个**高策略深度**的角色，需要玩家在"短期生存"和"长期发育"之间做出权衡。

**核心循环**: 养猪 → 能量上限 → 更强卡牌效果 → 投喂 → 收获 → 养猪/能量上限

**成功关键**:
1. 早期建立稳定的养猪/能量上限增长
2. 根据敌人类型选择投喂策略(单体堆高 vs AOE分散)
3. 把握收获时机，最大化里程碑奖励
4. 利用能量上限解锁强力卡牌效果

祝你养猪顺利，丰收满满！🐷
