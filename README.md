# RaisePig Mod

一个《杀戮尖塔》(Slay the Spire) 的自定义模组 - 养猪模组

## 简介

这是一个为《杀戮尖塔》游戏开发的自定义角色模组，名为"养猪"。

## 功能特性

- 自定义角色：养猪
- 自定义卡牌系统
- 自定义遗物
- 自定义能量系统

## 安装说明

1. 确保已安装《杀戮尖塔》和 ModTheSpire
2. 使用 Maven 构建项目：
   ```bash
   maven package
   ```
3. 构建完成后，jar 文件会自动复制到游戏 mods 目录

## 开发环境

- Java 1.8
- Maven
- Slay the Spire (Steam版本)
- ModTheSpire 3.30.3
- BaseMod

## 配置

在 `pom.xml` 中配置你的 Steam 安装路径：
```xml
<Steam.path>D:/steam/steamapps</Steam.path>
```
